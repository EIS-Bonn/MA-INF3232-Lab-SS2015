  بیرون و لابیش  خوشگله ولی طبقات و اتاق هاش خیلی قدیمی و کثیف بود و واس من قابل تحمل نبود تا اینکه مجبورشون کردم یک اتاق تو برجش بهم بدن که اونجا خیلی بهتر و ok بود.
