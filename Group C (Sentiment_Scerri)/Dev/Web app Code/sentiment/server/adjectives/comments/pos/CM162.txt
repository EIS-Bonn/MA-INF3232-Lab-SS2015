واقعا هتلی شیک لوکس و مدرنی بود وقتی جلو پنجره می ایستی تو اون ارتفاع و به منظره روبرو نگاه میکنی واقعا ارام بخشه...
  کافی شاپ واقعا شاد شلوغ و با منوی عالی بود.
  از همه بهتر مکان هتل که در بهترین نقطه شهر قرار داشت و جمعه صبح وقتی از هتل بیرون امدیم با صحنه باورنکردنی برخورد کردیم و اون صحنه جمعیت بسیار زیادی که واسه ورزش و پیاده روی امده بودن بود.
  از جمعه هتل استفاده کنید!
